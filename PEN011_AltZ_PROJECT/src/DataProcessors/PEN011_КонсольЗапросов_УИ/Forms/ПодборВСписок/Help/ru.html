Форма редактирования для типа контейнера "Массив" и "Список значений".

В подменю  можно выбрать вариант сохранения списка значений:

  • Скопировать в буфер обмена
  • Сохранить в файл XML

В подменю  можно выбрать вариант загрузки списка значений из различных источников:

  • Из буфера обмена
  • Из файлов на диске. Возможна загрузка из текстовых файлов, а так же из файлов XML.
